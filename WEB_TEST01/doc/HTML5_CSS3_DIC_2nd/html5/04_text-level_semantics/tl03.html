HTML이란 무엇입니까?

HTML란 HyperText Markup Language(하이퍼텍스트 마크업 언어)의 약자입니다. 태그라는 수단을 사용하여 텍스트에 문서의 구조나 장식 정보를 추가하고, 컴퓨터가 정보를 읽을 수 있도록 하는 언어를 말합니다. 이 언어를 사용하여 기술된 HTML 파일을 브라우저(웹 브라우저)에서 읽어들임으로써 웹 페이지로서 열람할 수 있게 됩니다.
현재 사용되고 있는 HTML의 버전은 4.01입니다. 이 4.01까지는 W3C(World Wide Web Consortium)라는 단체가 사양을 만들었습니다. 가장 새로운 HTML5은 현재 W3C와 WHATWG이라는 단체가 사양 책정을 진행중입니다.

브라우저란 무엇입니까?

웹 페이지를 열람하기 위한 소프트웨어를 말합니다. 웹 브라우저라고도 합니다.
대표적인 브라우저로는 Internet Explorer, Firefox, Opera, Netscape, Safari 등이 있습니다. 이런 일반적인 브라우저에서는 텍스트 외에 이미지나 동영상, 음악 등도 취급할 수 있지만, 텍스트밖에 표시할 수 없는 브라우저도 있습니다. 또한 각 브라우저가 독자적으로 확장한 태그, 독자성이 강한 기술 등을 이용하면 이를 지원하지 않는 브라우저에서는 페이지가 제대로 표시되지 않는 경우가 있습니다. 웹 페이지를 작성할 때에는 많은 브라우저가 존재한다는 것을 잊지 말기 바랍니다.

HTML 에디터란 무엇입니까?

HTML 편집 기능을 가진 에디터를 말합니다. HTML 파일은 텍스트 에디터만 있으면 작성할 수 있지만, HTML 에디터를 사용하면 텍스트 에디터에서 수작업으로 HTML을 기술하는 것보다 훨씬 손쉽게 HTML 파일을 작성할 수 있습니다.
시판하는 소프트웨어 외에도 네트워크 상에도 쉐어웨어나 프리웨어 등 수많은 HTML 에디터가 있습니다. 기능이나 조작 방법 등은 에디터별로 다르므로 자신에게 맞는 것을 선택하면 됩니다.
하지만 HTML 에디터를 이용하는 경우에도 다소의 HTML 지식은 필요합니다.

웹 페이지를 만들려면 무엇이 필요합니까?

기본적으로는 HTML 파일을 작성 및 확인하기 위한 텍스트 에디터와 웹 브라우저, 그리고 작성한 HTML 파일을 웹 페이지로서 공개하기 위한 프로바이더에 송신할 FTP 소프트웨어가 필요합니다.
웹 페이지에 이미지나 영상, 소리 등을 붙이고 싶은 경우는 이용하고 싶은 소재를 준비합니다. 시판 소재집이나 인터넷 상에서 소재를 제공하는 사이트에서 입수할 수 있습니다. 이런 데이터를 직접 편집해서 사용하고 싶다면 각 데이터를 편집할 수 있는 소프트웨어가 필요합니다. 필요에 따라 준비하기 바랍니다.
HTML 에디터에는 이런 각 툴이 갖고 있는 기능의 대부분을 겸비하고 있는 것도 있습니다.

어떤 이미지를 사용할 수 있습니까?

보통의 웹 페이지의 경우 일반적으로 GIF 형식, JPEG 형식으로 된 이미지를 사용할 수 있습니다. 비교적 새로운 브라우저에서는 PNG 형식의 이미지를 지원하는 것도 있습니다. 이미지의 크기는 페이지를 읽어들이는 속도와 관계가 있습니다. 데이터량이 많은 이미지는 가능한 한 사용하지 말기 바랍니다.
